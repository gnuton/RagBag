= Clojure =

== Basic stuff ==

=== Namespace ===
By default clojure uses "user" as namespace
You can set your own one with:
````
(ns antonio)
````

=== Comments ===
Comment lines start with ';'

=== Types ===

=== String ===
==== Concatenation ====
````
(str "ciao " "bello")
````

=== Math ===
````
; allowed operations +, -, *, /
(+ 1 1)

;
````
